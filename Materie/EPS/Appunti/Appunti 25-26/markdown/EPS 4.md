## Proprietà di $\mathbb{P}$
Siano $A, B, C \in p(\Omega)$ eventi. Allora:
1. Se $A \subseteq B$ allora $\mathbb{P}(A) \leq \mathbb{P}(B)$. Questo deriva direttamente dalla monotonia **non decrescente** della funzione $\mathbb{P}$
2. $\mathbb{P}(A \cup B) = \mathbb{P}(A) +\mathbb{P}(B) -\mathbb{P}(A \cap B)$ 
3. $\mathbb{P}(A \cup B) \leq \mathbb{P}(A) + \mathbb{P}(B)$ che nsegue direttamente dalla proprietà 2, visto che si toglie la sottrazione. Questa proprietà è nota ocme _subadditività_
4. $\mathbb{P}(A \cup B \cup C) = \mathbb{P}(A) +\mathbb{P}(B) +\mathbb{P}(C) -\mathbb{P}(A \cap B) -\mathbb{P}(A \cap C) -\mathbb{P}(B \cap C) +\mathbb{P}(A \cap B \ cap C)$
